Arrays / Objetos Strings

  • ValueOf()
    • - retorna o valor primitivo do objeto
  • charAt()
    • - retorna uma string contendo o caractere de posicao
  • concat (string, string, ..., n)
    • - esse método retorna uma string contendo string + string n
  • replace
    • - busca na string uma sub string que seja igual ao conteúdo
  • slice(início, fim)
    • - retorna a sub string contendo caracteres de incio ate o fim
  • split(separator, limit)
    • - retorna um objeto array contendo uma sub string entre o separator e limit (numero maximo de caracteres)
  • substring(inicio, fim)
    • - retorna o resultado da conersão do objeto que chamou o método (inicio, fim)
